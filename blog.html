Question-1: What are the different ways to select an element in the DOM?

There are several ways to select an element in the DOM(Document Object Model) These Are:

1.Tag Name(getElementsByTagName): in Javascript we can use Tag Name to access an element.For that we have to use getElementsByTagName('tagName').
02.By Id(getElementById): We can select an element using it's Id.For that we have to use getElementById('id') in js.
03.By ClassName(getElementsByClassName): We can select an element using it's ClassName.For that we have to use getElementsByClassName('ClassName') in js.This will provide us a html collection with all element that is using this ClassName.
04.By querySelector(querySelector): We can select an element by css selector for that we can use querySelector('selector').This will give us the first element using that selector.
05.By querySelectorAll(querySelectorAll): We can select an element by css selector for that we can use querySelectorAll('selector').This will give us all the element using and matching that selector.
These are the most used and popular way to select an element in the DOM.

Question-3: What is event delegation?

Event delegation is a technique that makes DOM event handling in Javascript more easier and efficient.Normally when we add a event handler it is directly associated with a specific dom element.But when there is too many child elements,then it is unnecessary to attach event handler for each elements and Here works event delegation,we adds the event handler to it's parents node and it's reduced unnecessary Event Handling.

Question-4: What is event bubbling in the DOM ?

Event Bubbling is a process in DOM where an event starts from a target element(where a event occurs such as a clicked or a interaction is happened) then it's starts to moves up(bubbling) to it's parent element untill it reaches the root element.
